Lottery Game

User and Computer numbers

The game is about the comparison of five numbers guessed by the user against five numbers generated by the computer. Clicking the Display numbers button shows both guessed and generated numbers. If there is matching, the user wins points described in the table. The Reset button reset the game.

Matching Numbers Award (pts)
Zero matching 0
One matching 10
Two matching 100
Three matching 1,000
Four matching 5,000
Five matching 10,000

Enter a number in five boxes from 1 to 20.
